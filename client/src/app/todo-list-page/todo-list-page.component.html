Системные требования

Минимальные требования к составу и параметрам технических средств: для стабильности работы информационной системы:
  • — объем оперативной памяти должен быть не менее 4 Гб;
  • — процессор с тактовой частотой 2,2 ГГц или большей;
  • — видеоадаптер, соответствующий стандарту DirectX 9 и поддерживающий минимальное разрешение экрана 926 x 572, и обладающий объёмом памяти не менее 512 мегабайт;
  • — версия операционной системы не ниже 7 Windows
Рекоминдуемые требования к составу и параметрам технических средств: для стабильности работы информационной системы:
  • — объем оперативной памяти должен быть не менее 6 Гб;
  • — процессор AMD A10-4600M 2300МГц
  • — видеоадаптер AMD Radeon HD 7670M
  • — монитор с разрешением 1366 x 768
  • — версия операционной системы не ниже 7 Windows

Пользователи

В данном приложение имеются две категории пользователей:
      Менеджер отдела продаж - являтся основным пользователем данной системы, имеет возможность управлением ассортимента (создание,удаление,редактирование) а также оформаление нового заказа. Данный пользователь может просматривать только свою историю заказов а также свою аналитику.
      Аналитик - являтся дополнительным пользователем который не может управлять ассортиментом и заказами но в свою очередь может просматривать аналитику и историю всех пользователей. В системе может быть только один аналитик, редактирование Email аналитика невозможна, пароль предоставлятся вместе с приложением.

Формы приложения

На основной форме расположена боковая навигация, информация о пользователе, кнопка для редактирования профиля пользователя а также кнопка для вывода курса валют на экран.

В основной форме для менеджера имеются следующие кнопки:

      Аналитика – открывает форму, в которой выводится графики по выручки и заказам;
      Обзор – открывает форму которая показывает динамику продаж за предыдущий день;
      История – открывает форму с историей заказов;
      Добавить заказ– открывает форму для добавления заказа;
      Ассортименты – открывает форму для добавления категорий и позиций;
      Документация – открывает форму с руководством по программе;

На форме «Аналитика» расположены два графика по выручки и по заказам. Период аналитики можно регулировать кнопками которые расположены наверху.

На форме «Обзор» расположены два блока которые показывают динамику продаж за предыдущий день.

На форме «История» выводится сведения о оформленных заказах, к которым можно применить фильтр.При нажатие на кнопку просмотра определенного заказа выводится окно с содержимым заказа

На форме «Добавить заказ» выводится категории товаров при нажатие на которых можно добавить определенные позиции к заказу.После добавления нужных позиций необходимо вызвать окно подтверждения заказа нажатием на кнопку «ЗАВЕРШИТЬ»

На форме «Ассортименты» выводится категории товаров находящиеся в системе, добавить категорию можно нажав на кнопку «Добавить категорию», при нажатие на категорию открывается окно редактирования где можно редактировать, удалить категорию или добавить определенные позиции к категории
